# **App Name**: Design Canvas

## Core Features:

- Project Gallery: Showcase design projects with high-quality images and customizable layouts.
- Detailed Project Pages: Dedicated pages for each project with in-depth descriptions, case studies, and additional media.
- About Me Section: A clean section to highlight designer's skills, experience, and contact information.
- Contact Form: An integrated contact form for potential clients to reach out directly.
- AI Content Refiner Tool: Utilize AI to suggest improvements for project descriptions and 'About Me' section content, ensuring conciseness and impact.

## Style Guidelines:

- Primary color: A sophisticated, subtle pastel purple (#A78AF0) to convey creativity and modern elegance.
- Background color: A very dark grey with a hint of purple (#141315), maintaining a sleek 'black and white' base as requested, ensuring text readability.
- Accent color: An analogous, slightly brighter pastel blue (#697EFF) to complement the primary color and highlight interactive elements.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable aesthetic, aligning with the minimalist and professional theme.
- Use minimalist line icons that maintain a clean and uncluttered visual language throughout the portfolio.
- Implement a grid-based layout with generous spacing and negative space to enhance focus on visual work and promote a premium feel.
- Subtle and smooth transitions for page changes and discreet hover effects for interactive elements to provide a polished user experience.
